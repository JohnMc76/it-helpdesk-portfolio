# Computer running slowly after startup

**Category:** Software  
**Priority:** Medium  

## Environment
- Operating System: Windows 10  
- Device Type: Laptop  
- Network Type: Home Wi-Fi  

---

## User Issue
User reports that the computer runs slowly after startup and is unsure of the cause.

---

## Initial Checks
- Confirmed issue occurs after every startup  
- Verified no recent hardware changes  

---

## Troubleshooting Steps
1. Opened Task Manager  
2. Reviewed startup applications  
3. Identified multiple unnecessary applications running at startup  
4. Disabled non-essential startup programs  

---

## Resolution
Issue resolved by disabling non-essential startup applications.

---

## Verification
User restarted the computer and confirmed improved system performance.

---

## Notes / Prevention
Advised user to restart the system weekly and limit unnecessary startup applications.

---

**Status:** Resolved
